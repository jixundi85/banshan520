## 🎯 关键算法清单

### 1. 诊断→推荐匹配算法
```javascript
// 输入：诊断答案数组
// 输出：企业标签 + 推荐OPC列表

匹配度 = Σ(标签匹配权重 × 技能匹配度 × 档期可用性 × 历史评分)
```

### 2. 需求→OPC匹配算法
```javascript
// 输入：需求标签、技能要求
// 输出：匹配OPC列表（按匹配度排序）

匹配度 = 技能重合度 × 0.4 + 行业经验 × 0.3 + 价格匹配 × 0.2 + 档期匹配 × 0.1
```

### 3. 项目状态机
```
待启动 → 进行中 → 待验收 → 已完成
   ↓         ↓         ↓
 已取消   已暂停   争议中
```

### 4. 信用分算法
```javascript
信用分 = 基础分 + 完成项目数 × 10 + 好评率 × 20 - 违约次数 × 30
```
